一、功能说明
html音乐播放器
1.调用网易云的api查询歌曲信息，api不是官方的，而是github上一个开源的项目https://github.com/Binaryify/NeteaseCloudMusicApi
2.这个项目是基于b站的一个教学视频做的，基本功能都是借鉴的，然后在上面加上了自己的功能
3.现在完成的功能有:
1)通过调用api搜索歌曲，歌词，评论，专辑
2)歌词滚动播放和逐字显示
3）专辑封面展示
